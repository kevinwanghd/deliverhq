TodoApp API

待办事项 CRUD 接口 — CR-EXAMPLE 示例

POST /api/todos

创建待办事项

GET /api/todos?page=1&size=20&status=pending

查询待办事项列表（分页 + 状态筛选）

PATCH /api/todos/{id}

更新待办事项

DELETE /api/todos/{id}

删除待办事项（软删除）

响应格式

{
  "id": 1,
  "title": "完成项目文档",
  "description": "编写 API 设计文档",
  "status": "pending",
  "due_date": "2026-06-10",
  "created_at": "2026-06-01T10:00:00Z",
  "updated_at": "2026-06-01T10:00:00Z"
}
