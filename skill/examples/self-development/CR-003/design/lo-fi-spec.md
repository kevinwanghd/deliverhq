## 交互流程

### 流程 1：查询列表
1. 用户选择筛选条件
2. 点击"查询"按钮
3. 表格刷新显示结果
4. 支持分页

### 流程 2：新增/编辑
1. 点击"新增"按钮 / 行操作"编辑"
2. 弹出表单弹窗
3. 填写必填字段
4. 点击"保存" → 校验 → 提交
5. 成功提示 + 关闭弹窗 + 刷新列表

### 流程 3：删除
1. 点击行操作"删除"
2. 弹出确认框
3. 确认 → 调用 API → 刷新列表
